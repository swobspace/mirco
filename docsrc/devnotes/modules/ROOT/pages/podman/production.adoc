== Handling Images


=== Update Ruby

.Update ruby base image
[source,sh]
----
podman pull docker.io/library/ruby:3.2.6-slim
----

=== Build Image

.Build
[source,sh]
----
podman build -t mirco .
----

.Inspect image
[source,sh]
----
podman run --rm -it mirco /bin/bash
----

=== Tag Image

.Tag image
[source,sh,subs="+attributes"]
----
podman tag mirco ghcr.io/swobspace/mirco:{mircoversion}
podman tag mirco ghcr.io/swobspace/mirco:latest
----

=== Push Image

.Push image
[source,sh,subs="+attributes"]
----
export CR_PAT=YOUR_TOKEN
echo $CR_PAT | podman login ghcr.io -u swobspace --password-stdin
podman push ghcr.io/swobspace/mirco:{mircoversion}
podman push ghcr.io/swobspace/mirco:latest
----
