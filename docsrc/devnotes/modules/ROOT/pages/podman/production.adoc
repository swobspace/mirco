== Handling Images

=== Build Image

.Build
[source,sh]
----
podman build -t mirco .
----

.Inspect image
[source,sh]
----
podman run --rm -it mirco /bin/bash
----

=== Tag Image

.Tag image
[source,sh]
----
podman tag mirco ghcr.io/swobspace/mirco:0.1.0
----

=== Push Image

.Push image
[source,sh]
----
export CR_PAT=YOUR_TOKEN
echo $CR_PAT | docker login ghcr.io -u USERNAME --password-stdin
docker push ghcr.io/swobspace/mirco:0.1.0
docker push ghcr.io/swobspace/mirco:latest
----
