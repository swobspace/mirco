== Add Extension

[source,bash]
----
bin/rails generate migration add_timescale
----

[source,ruby]
----
class AddTimescale < ActiveRecord::Migration[5.2]
  def change
    enable_extension("timescaledb") unless extensions.include? "timescaledb"
  end
end
----

[source,bash]
----
bin/rails db:migrate
----

NOTE: Die Migration setzt voraus, dass die Shared Libriaries von TimescaleDB in postgresql.conf eingetragen sind und PostgreSQL neu gestartet wurde. Siehe auch xref:mirco:installation-guide:database-timescale.adoc[].


.Check extension
[source,bash]
----
echo "\dx" | rails dbconsole
----
